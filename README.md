# 학내연주 평가지 자동 취합기

구글 폼 평가 결과를 연주자별 PDF로 정리해서 ZIP 파일로 내려받는 Streamlit 앱입니다.

## 핵심 정리

- `CSV는 필수`입니다.
- `HTML은 선택`입니다.
- HTML 대신 공식 연주자 명단을 직접 붙여넣어도 됩니다.

이유:

- CSV에는 실제 점수와 익명 피드백 원본이 들어 있습니다.
- HTML은 연주자 순서와 이름을 자동 추출할 때만 사용합니다.

## 사용 방법

1. 가상환경 생성 및 패키지 설치

```bash
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
```

2. 앱 실행

```bash
.venv/bin/streamlit run app.py
```

3. 브라우저에서 사용

- 방법 A: `HTML에서 자동 추출`
  - 구글 폼 HTML 사본 업로드
  - 응답 CSV 업로드
- 방법 B: `명단 직접 입력`
  - 공식 연주자 명단을 한 줄에 한 명씩 붙여넣기
  - 응답 CSV 업로드
- 선택: `검증용 기준 명단`
  - 공식 명단이나 학사 명단을 붙여넣으면 오타 후보와 누락 이름을 자동 비교

4. `PDF 파일명용 학내 날짜` 확인

예: `2026-04-08`

5. `PDF 자동 생성 시작` 클릭

6. `PDF ZIP 다운로드` 클릭

## 출력 파일 형식

- 개별 PDF 이름: `학내날짜_대상자_평가요약.pdf`
- 예시: `2026-04-08_최광_평가요약.pdf`

## 오타 방지 팁

현재 앱은 연주자 이름을 응답자가 직접 적는 구조가 아니라, 연주자 명단 순서와 CSV 열 순서를 맞춰서 처리합니다.

그래서 `최광`을 `최굉`처럼 잘못 적는 문제는 현재 구조에서는 크게 발생하지 않습니다.

또한 선택 입력인 `검증용 기준 명단`을 넣으면 현재 명단과 비교해서 다음 정보를 화면에 보여줍니다.

- 현재 명단과 기준 명단의 정확 일치 수
- 현재 명단에서 수동 확인이 필요한 이름
- 한 글자 차이 또는 유사도 기반 추천 후보

만약 앞으로 이름 직접 입력형 폼으로 바뀐다면 다음 순서를 추천합니다.

1. 공식 명단 기준값 확보
2. 공백/특수문자 정리 후 비교
3. 유사도 매칭으로 자동 보정 후보 제시
4. 애매한 경우 관리자 확인
